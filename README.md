# Cities-Board Project

Cities-Board project aims to automate the development of graphic dashboards for smart cities. This project is developed in the Systems Engineering program at the University of Nariño, Pasto, Nariño, Colombia. 

This repository contains the development environment of Cities-Board. If you are interested on using Cities-Board to model and implement your city's dashboard, you should visit our [cities-board-plugin](https://github.com/cabrerac/cities-board-dev) repository.
